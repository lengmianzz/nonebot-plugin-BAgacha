### 功能:
 - 自动更新抽卡概率
 - 自动更新up卡池, 检测更新卡池和图标库
 - 展示当前UP学生
 - 展示当前概率


### 安装:
 - 使用 nb-cli 安装  
```
nb plugin install nonebot-plugin-badrawcard
```

 - 使用 pip
```
pip install nonebot-plugin-badrawcard
```

### 配置:
 - proxy: 本插件需要使用代理(`http://ip:host`格式)
 - redis_host: Redis的host(默认为localhost)
 - redis_port: Redis的开放端口(默认为6379)  


### 触发:
 - `/ba单抽`
 - `/ba十连`
 - `/ba来一井`
 - `/当前概率`
 - `/当前up`
 - `/订阅更新`(当自动更新up池时提醒用户)
